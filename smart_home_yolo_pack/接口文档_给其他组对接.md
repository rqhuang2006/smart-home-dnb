# 智能家居 YOLO 模块接口文档

服务默认地址：

```text
http://<香橙派IP>:5000
```

如果在香橙派本机访问：

```text
http://127.0.0.1:5000
```

---

### 请求

```http
GET /health
```

```json
{
  "status": "ok",
  "model_path_exists": true,
  "database": "/home/orangepi/smart_home_yolo_pack/results/detections.db",
```

用途：GUI组/后端组判断YOLO服务是否启动。

## 3. 上传图片并识别

### 请求

```http
POST /api/detect/upload
Content-Type: multipart/form-data
```

字段名：

```text
image
```

### curl 示例

```bash
curl -X POST -F "image=@test.jpg" http://127.0.0.1:5000/api/detect/upload
```

---

## 4. 获取最新识别结果

### 请求

```http
GET /api/latest
```

用途：GUI定时刷新最新结果。

---

## 5. 获取识别历史

### 请求

```http
GET /api/history?limit=20
```

`limit` 最大建议不超过 200。


```http

```

### TXT

```http

```

## 7. 给硬件组的对接约定

YOLO模块可以在检测到指定类别后，向硬件组接口发送POST请求。

配置位置：

```text
config.json -> hardware
```

示例配置：

```json
{
  "enabled": true,
  "timeout_seconds": 2,
  "trigger_rules": [
    {
      "name": "检测到灯泡后开灯",
      "classes": ["bulb", "light_bulb", "灯泡"],
      "min_confidence": 0.4,
      "method": "POST",
      "endpoint": "http://127.0.0.1:5001/light/on",
      "payload": {
        "device": "light",
        "action": "on"
      }
    }
  ]
}
```

当检测到灯泡后，YOLO模块会向硬件接口发送类似数据：

```json
{
  "device": "light",
  "action": "on",
  "trigger_rule": "检测到灯泡后开灯",
  "detection_id": 1,
  "time": "2026-07-06 16:55:02",
  "detected_objects": [
    {
      "class_id": 2,
      "class_name": "bulb",
      "confidence": 0.83
    }
  ]
}
```

硬件组只需要接收这个POST请求，然后控制GPIO、继电器、LED或其他模块即可。

## 8. 需要其他组提供的信息

### 硬件组需要提供

1. 是否由YOLO模块直接控制GPIO，还是调用硬件组HTTP接口。
2. 如果是HTTP接口，需要提供接口地址，例如：

```text
http://硬件服务IP:端口/light/on
```

3. 如果是GPIO，需要提供具体引脚号、有效电平、控制逻辑。

### GUI/后端组需要确认

1. 是否直接使用本模块自带网页。
2. 是否需要把识别结果同步到统一后端数据库。
3. 是否需要定时自动检测，而不是点击按钮检测。

如果需要定时自动检测，可以在GUI组前端定时调用：

```http
POST /api/detect/camera
```

或者在 `app.py` 中增加后台定时任务。

# 小组兼容接口：POST /api/v1/vision/detect

本版本新增小组兼容接口，用于对接后端/GUI。

请求：multipart/form-data

字段：

| 字段 | 说明 |
|---|---|
| source | upload 或 camera，默认 upload |
| image | source=upload 时上传图片 |

返回核心字段：

| 字段 | 说明 |
|---|---|
| detections[i].label | 类别名 |
| detections[i].confidence | 置信度 |
| detections[i].bbox | 检测框坐标 |
| light_bulb_detected | 是否检测到灯泡 |

测试命令：

```powershell
curl.exe -X POST http://127.0.0.1:5000/api/v1/vision/detect -F "source=upload" -F "image=@C:\Users\34668\Desktop\test.jpg"
```
